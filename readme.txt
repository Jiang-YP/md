项目目的：计算管壳式膜精馏模组中DCMD过程（SCUT2014届毕业设计）

开发人员：关国强（PM），张善俊（化2班），罗伶（化3班）

通过指定入口条件（温度、压力和流率）和膜组参数（膜性质和尺寸、流道尺寸等）计算出口条件及膜组性能参数（平均渗透通量、热效率等）
求解控制方程组（常微分方程组）和微元衡算两种计算方案
采用Fortran语言，Visual Studio 2008开发环境（采用固定格式）编写计算代码，最终编译为供Aspen Plus调用的dll

目录结构:
.\bin  - 二进制文件(例如生成的dll文件等)
.\dev  - Visual Studio开发目录
.\doc  - 文档
.\etc  - 其他的配置文件(例如编译脚本)
.\font - 字体
.\src  - 源代码
.\tmp  - 编译过程临时文件
